# 학생 발명품 유사도 검토 툴

학생 발명품 설명서 PDF를 업로드하면 AI가 자동으로 분석합니다.

## 제공 기능

1. 핵심 아이디어 요약
2. 특허·수상작·시판 제품 검색 키워드 생성
3. 기존 선행 자료와의 유사 가능성 분석
4. 항목별 유사도 점수 (해결 문제 / 핵심 기능 / 구조 / 작동 원리 / 사용 상황)
5. 차별점 분석 및 보완 지도 방향 제시
6. 심사자 참고용 최종 보고서

## 로컬 실행

```bash
npm install
npm run dev
```

## Netlify 배포

1. 이 저장소를 Netlify에 연결
2. Build command: `npm run build`
3. Publish directory: `dist`

## 사용 방법

1. [Anthropic Console](https://console.anthropic.com/settings/keys)에서 API 키 발급
2. 앱 상단에 API 키 입력 (브라우저 로컬에만 저장됨)
3. 발명품 설명서 PDF 업로드
4. "유사도 분석 시작" 클릭 (30~60초 소요)
5. 결과 확인 후 "보고서 인쇄"

## 주의사항

- 스캔 이미지 PDF는 텍스트 추출이 불가능합니다 (텍스트 레이어 있는 PDF만 지원)
- 본 분석은 법적 특허 침해 여부를 판단하지 않습니다
- 유사 가능성 수준의 참고 정보이며 최종 판단은 심사자가 해야 합니다
